Simutrans 128x128 Home

Welcome! This page is home to 128x128 pixel version of the wonderful Simutrans game, usually called colloquially pak128. Simutrans is a free & open source transport economic simulation game, available for a multitude of operating systems and computer types.

Latest pak128 version is 1.4.6, compatible with Simutrans 102.2 and later versions. Announcement on forum includes changelog and some more comments.


(sourceforge)

Further information

Please remember: if you want to install the game, you will need 128x128 base package + Simutrans executable for your operating system. Don't forget to download that, too!

Currently there exist also other image sets for Simutrans, bringing really large variety to the game. You can learn more about them on page about paksets.

Are you missing an object that was in old versions, but has been removed, such as a factory? Can't load old savegame? Try the compatibility pack which includes backup of these objects. Older releases are available on sourceforge, too.

Nightly builds of pak128 are available at nightly page. These are snapshots of development in between versioned releases, and may be broken or incompatible - on the other hand, they might include new features as well. Remember that for nightly pak you need nightly of the program, too. They are available on the same page. For information about recent improvements, try the web archive of development logs.

Want to paint some addons for this pakset? Graphic archive has lots of templates and helpful parts, and on Simutrans wiki should be enough to get you started.

Last source export was at revision 1111, available as a zip file (60 MB) and on GitHub. These materials are redacted because there are parts that authors wished to remain private. We are working to replace these parts and achieve open source status. For now, the exports include all important material that an author could need as a reference, even after redaction.
